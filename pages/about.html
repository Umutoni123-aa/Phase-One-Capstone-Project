About Book Explorer

What is Book Explorer?

Book Explorer is a cozy web app that helps book lovers search and discover books using a free Open Library API. Browse a wide collection of titles, save your favourites, and keep track of books to read later.

How it works

  • Search for any book by title using the search bar on the home page.
  • Click Add to Favourites on any book card to save it.
  • Visit the Favourites page to see your saved books.
  • Your favourites are saved in your browser and available whenever you return.

Built With

  • HTML5 and Tailwind CSS for responsive design
  • JavaScript ES6 modules for interactive features
  • Open Library API for book data
  • localStorage to save user favourites
